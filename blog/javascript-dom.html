What's the difference between HTML and CSS?

HTML is like a movies script. It contains the content and substance with which we consume. CSS is like the movies editing. It determines how it looks.

What is 'control flow' and 'loops'?

Control flow is the order in which the computer executes your code. It's like making a sandwich. First we lay out our bottom piece of bread, then we spread the butter, then we add our various ingredients. Lastly, we add the top layer of bread to complete the sandwich. The order in which the sandwich is made is like the code that needs to be executed by the control flow.

Loops are sections of code that will repeat until certain conditions are met. They are like reps in exercising. If we want to do 20 push-ups then since each push-up is the same we would loop through the process of 1 push-up 20 times.

What is the DOM?

DOM stands for Document Object Model. The DOM can be viewed as the roots of a tree, where each element of the DOM branches off from a node above. An example of interacting with the DOM would be when accessing a CSS style sheet from your HTML.

          
<link rel="sylesheet" type="text/css" />

The code must go back along its branch to reach the source of the HTML and then find the CSS style sheet. Doing so will link the two separate files together.

What is the difference between accessing data from arrays and objects?

An 'Object' is a single data structure, while an 'Array' is multiple objects in an ordered collection. When we access data from an array we must first access an object within the array before we can access data from an object to use or maniplulate.

What are functions?

A 'Function' is an isolated block of code the can be called to run from scripts. A create example of the use of functions is pressing a button. When we press a button we want something to happen. Whatever happens is a set of instructions in code that will only run when the button is pressed.

Another use of functions is that the same section of code can be run while receiving different variables. As an example, if we have two buttons that change the color of a circle on screen, the function will change the color while each button can request what color they want the function to use.
