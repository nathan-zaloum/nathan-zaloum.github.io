"What is the difference between Margin, Border, and Padding?"

The margin is the space around an element's border, padding is the space between an element's border and it's content, and the border is what sits between the margin and padding. The border is by default 0 px and so is not seen by default.

Margins will change the elements position on the page. When there are multiple elements margins determine the "whitespace" between them. Here is an example of code that changes the margin of an element in HTML:

< style = "margin-bottom: 10px;" />

The border is the edge of the element and is 0px by default, which is why it is not seen initially. Here is an example of code that changes the border of an element in HTML:

< style = "border: 2px;" />

Padding is between the content and the border of the element. Fixed padding can force the content to enlarge as the element is enlarged. Here is an example of code that changes the padding of an element in HTML:

< style = "padding-left: 10px; padding-right: 10px;" />

< style = "padding-top: 15px; padding-bottom: 15px;" />
